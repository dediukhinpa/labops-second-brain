# 10-strategy

Long-term mission, milestones, north-star artifacts.

## What goes here

- Mission statement and stable principles
- Yearly / quarterly milestones with success criteria
- North-star metrics (one or two numbers that matter)
- Strategic bets and the reasoning behind them
- Org chart, agent roster, role definitions
- Vision documents that span multiple projects

## What does NOT go here

- Active project status — that lives in `40-projects/`
- Tactical decisions about a single feature — `30-decisions/`
- Day-to-day progress notes — `20-daily/`

## Typical authors

- The owner (you), when shaping direction
- `coordinator-agent`, when synthesizing across projects

## Cadence

- Add: quarterly or when a major directional change happens
- Read: weekly skim by every agent at session start
- Update: rarely — strategy that changes weekly is not strategy
